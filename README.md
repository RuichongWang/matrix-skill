# matrix-skill

A [Claude Code](https://claude.ai/claude-code) skill that downloads domain knowledge on demand — search the web, synthesize expertise, save it as a permanent skill.

Like The Matrix: you think of a domain, you know it.

## What it does

```
/matrix <domain> [level]
/matrix <domain> <subtopic> [level]
```

| Example | What happens |
|---------|-------------|
| `/matrix biology` | Broad survey of biology — cells, DNA, evolution, energy |
| `/matrix biology 2` | Same, at introductory level (plain English, no jargon) |
| `/matrix biology photosynthesis 9` | Deep dive on photosynthesis at research level |
| `/matrix patent-law` | How patent law works — claims, prior art, prosecution |
| `/matrix medium-blog-writing 7` | What actually drives distribution and earnings on Medium |

**Level 1–10** (default: 5):
- `1–3` — introductory, accessible to anyone
- `4–6` — mechanisms, named models, current consensus
- `7–8` — research-level, recent findings, active debates
- `9–10` — cutting edge, open problems, frontier knowledge

Skills are saved to `~/.claude/skills/<domain>/` and load automatically in future conversations.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/RuichongWang/matrix-skill/main/install.sh | bash
```

Or manually:

```bash
mkdir -p ~/.claude/skills/matrix
curl -fsSL https://raw.githubusercontent.com/RuichongWang/matrix-skill/main/.claude/skills/matrix/SKILL.md \
  -o ~/.claude/skills/matrix/SKILL.md
```

## How skills accumulate

```
~/.claude/skills/
  matrix/SKILL.md                    ← the meta-skill (this repo)
  biology/SKILL.md                   ← broad survey, level 5
  biology/photosynthesis.md          ← deep dive, level 9
  patent-law/SKILL.md
  medium-blog-writing/SKILL.md
  options-trading/SKILL.md
```

Skills upgrade too — run `/matrix biology 8` on an existing level-5 file and it enriches the existing content rather than starting over.
